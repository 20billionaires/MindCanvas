## Flow Strategy - GitHub Flow

- main : 정식 배포된 버젼을 담고 있는 브랜치
- develop : 현재 개발이 완료된 상태와 일치하는 브랜치
- feature : 기능 개발을 위한 브랜치로, 기능 구현이 완료되면 삭제되는 브랜치
  - ex. design/issue_320 : (커밋컨벤션명)/issue_(이슈번호)
